Cada elemento HTML puede llevar el atributo id. Se utiliza identificar de forma ùnica ese elemento de otro elementos de la pagina.Su valor debe comenzar con una letra o un guion bajo (no un nùmero o cualquier otro caracter). Es importante que no ha dos elementos en la misma pagina tienen el mismo valor para sus atributos ID (de lo contrario, el valor ya no es ùnico)

dar a un elemento de identidad ùnica le permite darle un estilo diferente que cualquier otra instancia del mismo elemento en la pagina.

Por ejemplo, es posible que desee asignar un parrafo dentro e la pagina (quizas un parrafo que contiene una cotizaciòn de extracciòn) un estilo difenrente que todos los demas pàrrados. En el ejemplo, el parrafo con el atributo ID cuyo valor es pullquote se hace en mayùsculas usando CSS.
